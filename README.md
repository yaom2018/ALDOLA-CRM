#### 2、需要安装的依赖包：

 wepy: npm install wepy-cli -g
 nodeJs安装及配置相关环境

运行方式：

```
运行 npm install 安装所有依赖项
运行 wepy build --watch 命令，开启 wepy 项目的实时编译功能
打开微信开发者工具，加载 wepy 项目并查看效果
```
